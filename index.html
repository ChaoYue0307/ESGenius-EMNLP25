Dataset

Expert ESG knowledge, packaged for repeatable model evaluation.

The benchmark covers sustainability reporting, climate disclosure, biodiversity, energy, governance, and standards-driven ESG reasoning across IPCC, GRI, SASB, ISO, IFRS/ISSB, TCFD, and CDP sources.

Evaluation Code

Three evaluation paths, one shared utility layer.

Open-source models

Run Hugging Face causal language models locally with deterministic generation settings and batch-level result saving.

python eval_opensource.py --dataset ESGenius_1136q.csv

Reference-aware RAG

Use the reference CSV to prepend source snippets before generation, keeping retrieval behavior simple and inspectable.

python eval_opensource_rag.py --dataset ESGenius_w_ref_1136q.csv

Qwen API

Evaluate Dashscope-compatible Qwen models through the API path with retry handling and Excel reporting.

python eval_qwen_api.py --dataset ESGenius_1136q.csv
